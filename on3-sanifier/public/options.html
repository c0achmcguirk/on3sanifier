Rating Threshold

The rating threshold controls which posts are showing on the screen. The higher the number, the fewer posts that will be shown. on3 Sanifier will hide any posts that have fewer upvotes than the rating threshold you choose below.

Ignored Threads

Ignored individual threads. If it's listed below, on3 Sanifier will not show it in thread listings.
Ignored threads that contain
filter_alt
If you add a string to this list, it will ignore threads that contain the string in the title. NOTE: It won't ignore threads that have the string in the one of the posts in the thread, it just looks at the title.
Favorite on3 Page
favorite
Your favorite page on on3. It will make the button on the extension's popup open this page.
Debug Mode
bug_report
When enabled, the extension will print detailed debugging information to the browser's console.
